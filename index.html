My recent projects.

Chirpy.

Chirpy is a chrome extension that takes the toxicity out of the internet, but in particular Twitter.

It works in tandem with the Watson API developed by IBM to detect negative tones, for example; anger, fear or sadness related content on someones Twitter timeline and then automatically removes said negative content.

It also has a manual setting where users can input a particular word or phrase and those will also be removed!

This was a project I worked on as part of a team in the Le Wagon Bootcamp, where we had to pitch, plan and then develop a project within a short time period.

As a team we worked through Chirpy and had a total blast along the way, although naturally there was some issues when working with Twitter data and the way that the API worked, in the end we managed to prevail over the challenge and Chirpy was presented at our graduation / demo day.

2Bridal

Two Bridal is a marketplace that allows users worldwide to list or rent dresses for others to use for their own special day.

During this project I was in charge of all of the front end aspects and I think that it turned out really well with the amount of time that we had to work on it.

It was a great experience to be able to work with some JS libraries that added some pizzaz to the page where users can see all of the details about a dress.

Some of the challenges during this particular project came from using a library that was a little outdated, unfortunately in the end I ended up having to switch to another to get the project working smoothly, but I think it looked amazing in the end!
